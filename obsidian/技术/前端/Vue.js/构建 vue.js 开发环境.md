# 构建 Vue.js 开发环境

## 安装 node.js 
点击: http://nodejs.cn/

## 国内安装 cnpm
    npm install -g cnpm --registry=https://registry.npm.taobao.org

## vue-cli 脚手架安装
    npm install vue-cli -g
    
## 初始化 vue 项目
    vue init webpack vue-demo
    
## 安装依赖
    cnpm install

## 运行执行
    cnpm run dev

> Vue.js 学习 请看其官方[文档](https://cn.vuejs.org/v2/guide/)
