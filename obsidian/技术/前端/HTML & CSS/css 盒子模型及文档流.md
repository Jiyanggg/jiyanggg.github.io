


#### 盒子的组成

![image](CE8FF37DCC2C4B11B8FEBD9CE629D0AA)

```
content 内容
padding 内边框：内容距离边框的距离
border  边框
margin  外边框：边框到“父元素”或“兄弟元素”之间的那一部分

```


#### 内边框 （补白）

```
padding-top:20px;
padding-right:40px;
padding-bottom:60px;
padding-left:80px;

简写：(顺时针指定)
padding: 1px
padding: 1px 2px
padding: 1px 2px 1px 2px 
```



#### 边框

```
border-width
border-style
border-color

border: 1px solid red;
```


#### 宽高

display 属性可以实现行内元素和块元素的互换

```
块元素，可以设置 width / height
行内元素，宽高只由内容决定
```
