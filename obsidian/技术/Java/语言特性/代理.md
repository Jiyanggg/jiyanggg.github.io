用来不改变原对象的基础上, 增强对象的功能, 类似于 Python 的装饰器. 有两种方法.

### 静态代理

使用 Java 接口的多样性来实现, 接口中同一个方法的不同表达 (属于一种设计模式 Proxy Mode)


### 动态代理

#### 1. JDK 动态代理

关键词 invoke, 但是只能代理实现了接口的类

#### 2. CGLIB 代理

关键词 enhancer, CGLIB 可以代理未实现任何接口的类
